## Features
Hugo sections available in theme and admin

* pages
* persons
* offices
* posts
  * authors (taxonomy)
  * categories (taxonomy)
  * tags (taxonomy)
* projects
  * tags (taxonomy)
  * types (taxonomy)
* publications
  * concerned (taxonomy)
